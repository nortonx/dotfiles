# Code Review from GitHub PRs

Review code from Github PR for the currant branch and implement suggested changes

## Usage

/commandname [arguments]

## Tasks

1. Use gh client to read and fetch @copilot automated reviews
2. Read the review content and evaluate against codebase, following best practices,
and coding standards defined in the project's CLAUDE.md
3. Show a report with a plan, with your suggestions
4. After developer approval, implement the suggested reviews and comment them as
a reply on each of their respective thread
5. Commit code, create a sucint message with technical description

## Output

A report with the changes implemented, summarizing code improvements
